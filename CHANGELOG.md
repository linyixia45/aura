# Changelog

## [0.1.0] - 2025-02

### 新增

- Vue 式响应式 API：`ref`、`computed`、`reactive`、`effect`
- 模板指令：`{{ }}`、`v-if`、`v-show`、`v-for`、`v-model`、`:class`、`:style`、`@event`
- 事件修饰符：`@keydown.enter`、`@keydown.esc`、`@keydown.tab`、`@keydown.space`
- 生命周期：`onMounted`
- 响应式工具：`watch`、`watchEffect`、`toRefs`、`nextTick`
- 反 AI 味设计令牌与基础组件
- XSS 转义（`{{ }}` 插值）
- `npm run build` 生成压缩版
