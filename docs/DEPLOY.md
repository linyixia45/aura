# Aura 部署指南

## GitHub Pages

### 第一步：配置 Pages 发布源

1. 打开 https://github.com/linyixia45/aura/settings/pages
2. 在 **构建和部署** 下，**Source** 选择 **从分支部署**
3. **Branch** 选 **gh-pages**，**Folder** 选 **/ (root)**
4. 点击 **保存**

### 第二步：推送或手动运行工作流

- 推送到 `master` 或 `main` 后会自动部署到 gh-pages 分支
- 或到 https://github.com/linyixia45/aura/actions 手动运行 **Deploy to GitHub Pages**

### 部署地址

- https://linyixia45.github.io/aura/
